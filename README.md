## Description

A simple Todo app written in Nest.js

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ npm run test

## Docs

Generate Docs for Whole Project Code

$ yarn docs
